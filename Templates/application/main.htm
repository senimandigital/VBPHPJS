Selamat datang di VBPHPJS

Jika ingin membuat Aplikasi Desktop Opensource yang perlu kita lakukan hanyalah membuat sebuah Form dan itu terserah mau mengunakan Basic, C, Java, Pascal, Python itu semua bisa sekali. Kita hanya perlu mengaktifkan satu kontrol WebBrowser kemudian mengcompile program tersebut, dengan demikian Main Aplikasi tidak akan mencapai 100 Kb.

Aplikasi yang sedang anda buka sekarang kami beri nama VBPHPJS, diberi nama seperti itu karena aplikasi ini mampu menjalankan Printah dari Visual Basic, PHP, Javascript dan VBScript sekaligus, tidak hanya itu karena termasuk disini adalah Html dan CSS tentunya. aplikasi ini menggunakan Browser yang sudah terinstal pada komputer Anda. itu yang membuatnya menjadi sangat ringan.

Kami berbeda dari pengelola Opensource lain, yang mungkin aktif melakukan update Produk-produk mereka, sedangkan kami memilih langkah lain, kami menyediakan teknologi pengembangan untuk setiap Opensource yang kami tulis.

Apabila anda juga tertarik mengembangkan aplikasi desktop berbasis PHP anda bisa membuat folder dimana anda meletakan file ini sebagai folder php, untuk melakukannya anda bisa melakukan itu dengan menyertakan php portable, tutorial untuk membuat webserver php portabel banyak tersedia di internet, bahkan beberapa paket webserver sudah menyertakan fasilitas untuk membuat php portable.
